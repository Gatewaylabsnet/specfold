# Roadmap

## MVP

- Import OpenAPI/Swagger from pasted text.
- Edit imported and manual requests.
- Manage local environments.
- Send variable-resolved requests from Electron main.
- Export collection or selected folders.
- Package script for portable Windows exe.

## Next

- Import Collection JSON from a file picker.
- Add drag-and-drop request and folder reordering.
- Add response history with secret-safe storage settings.
- Improve schema rendering for complex OpenAPI bodies.
- Add unused-component pruning for exports.
- Add OS credential vault support for secret variables.
- Add installer polish, code signing, and smoke tests for packaged builds.
